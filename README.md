**ğŸ“Œ Project Overview**



This project focuses on the design and deployment of a small-scale network security laboratory using a pfSense firewall and a T-Pot honeypot in a virtualized environment. The primary objective is to observe and analyze cyberattack behaviors while ensuring the security of the internal network through network isolation and strict firewall policies.



The system is designed to fully separate the internal network from the honeypot network to reduce the risk of attack propagation, while still allowing controlled exposure of selected services for attack data collection.
